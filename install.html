앱 주소: hyessung0806.github.io/tongdok-bible
이 주소를 가족과 공유하세요
설치하면 좋은 점 — 홈 화면 아이콘으로 전체화면 실행, 더 빠르고, 진도가 안전하게 보관됩니다. 아래에서 내 기기를 찾아 따라 하세요. (1~2분이면 끝나요)

📱 아이폰 (iPhone)

반드시 Safari(사파리)로 여세요 — 크롬 앱으로는 설치가 안 됩니다.

  1. Safari로 위 앱 열기 또는 앱 주소로 접속합니다.
  2. 화면 아래쪽 가운데의 공유 ⬆️ 버튼을 누릅니다.
  3. 목록을 아래로 내려 홈 화면에 추가를 누릅니다.
  4. 오른쪽 위 추가를 누르면 끝! 홈 화면의 통독 아이콘으로 실행하세요.

📲 아이패드 (iPad)

아이폰과 같지만 공유 버튼이 화면 오른쪽 위에 있어요. (Safari로 열기)

  1. Safari로 앱 주소에 접속합니다.
  2. 화면 오른쪽 위의 공유 ⬆️ 버튼을 누릅니다.
  3. 홈 화면에 추가를 누릅니다.
  4. 오른쪽 위 추가 → 홈 화면 통독 아이콘으로 실행.

🤖 안드로이드 폰

Chrome(크롬) 또는 삼성 인터넷으로 여세요.

  1. Chrome으로 위 앱 열기 또는 앱 주소로 접속합니다.
  2. 아래에 뜨는 홈 화면에 추가 / 설치 배너를 누릅니다.
  3. 배너가 없으면 오른쪽 위 ⋮ 메뉴 → 앱 설치(또는 홈 화면에 추가).
  4. 한 번 더 설치/추가 → 홈 화면 통독 아이콘으로 실행.

🖥️ 갤럭시탭

안드로이드 폰과 같아요. 삼성 인터넷 또는 Chrome으로.

  1. 삼성 인터넷(또는 Chrome)으로 앱 주소에 접속합니다.
  2. 삼성 인터넷: 아래 ≡ 메뉴 → 현재 페이지 추가 → 홈 화면.
  3. Chrome: ⋮ 메뉴 → 앱 설치 / 홈 화면에 추가.
  4. 홈 화면의 통독 아이콘으로 실행. (가로/세로 모두 보기 편해요)

설치 후 알아두면 좋아요

📵 인터넷 없이 읽기 (지하철·비행기)

설치 후 설정 ⚙️ → "전체 본문 받기"를 한 번 누르면 성경 66권을 미리 받아, 인터넷이 없어도 모든 회차를 읽을 수 있어요. (와이파이에서 1회 권장)

🔁 부부·기기끼리 진도 맞추기

진도는 기기마다 따로 저장돼요. 옮기려면 설정 → "진도 내보내기"로 코드를 복사(또는 파일 저장)해서, 다른 기기 "진도 가져오기"에 붙여넣으면 됩니다. (읽은 회차가 합쳐져요)

✓ 읽음 체크 & 자동 업데이트

그날 말씀을 읽고 "읽음 완료"를 누르면 통독표에 도장이 찍히고 다음 회차로 넘어가요. 앱은 온라인으로 다시 열 때 자동 업데이트됩니다(재설치 불필요).

✍️ 태블릿 필사 (애플펜슬·S펜)

아이패드·갤럭시탭(또는 S펜 폰)에서 읽기 화면 오른쪽 위 ✍️ 필사하기를 누르면, 오늘 말씀을 펜으로 화면 한가득 따라 쓸 수 있어요. 따라쓰기(흐린 글씨 위)↔보고 쓰기(빈 줄), 🖊️펜↔✒️만년필(필압) 굵기를 버튼으로 고르고, ◀ ▶로 쪽을 넘깁니다. 지우개는 화면 아래 지우개 버튼으로 켜며, 빨간 원이 펜을 따라다니며 닿은 획을 통째로 지웁니다.

쓰던 글씨가 그대로 남아요. 중간에 나갔다 다시 들어오면 쓰던 데서 이어서 쓸 수 있고, 지난 필사도 그 회차를 다시 열면 보입니다(이 기기에 저장). 필사 완료 버튼은 그 쪽을 충분히 써야 켜지고(덜 썼으면 “더 써주세요 %”로 표시), 다 못 채워도 ◀ ▶로 넘어갈 수 있어요. 기기를 바꾸거나 백업하려면 설정 → 필사 글씨 내보내기(파일)를 쓰세요. (S펜 측면 버튼·애플펜슬 더블탭으로 지우개 켜기는 웹 제약으로 안 될 수 있어요 — 지우개 버튼을 쓰세요.) 손바닥이 닿아도 화면이 움직이지 않습니다.
